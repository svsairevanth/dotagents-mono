# Glossary

## Task
A user-scoped unit of desired work.

## Run
A concrete execution attempt for a task.

## Step
A meaningful stage within a run.

## Action
A specific operation taken by an agent or tool.

## Artifact
A tangible output produced by the run.

## Review
The acceptance surface where results are inspected.

## Approval
A required user decision before the system may continue.

## Blocker
A condition preventing forward progress.

## Steering
Natural language or structured input used to redirect a run without fully restarting it.

## Autonomy level
How independently the system may act during the run.

## Verification
Evidence that the output or execution succeeded.
